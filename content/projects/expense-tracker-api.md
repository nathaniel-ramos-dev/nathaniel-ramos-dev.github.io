An expense tracking application built with Go that provides a RESTful API alongside a lightweight server-rendered web interface. Users can create accounts, securely authenticate, and manage their personal expenses while tracking their spending through a simple and responsive interface.

The project was built to explore modern backend development in Go, emphasizing clean architecture, type-safe database access, secure authentication, and minimal JavaScript by leveraging server-side rendering with HTMX.
